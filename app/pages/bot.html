Como capturar o ID correto?

No WhatsApp Web, selecione a conversa e use o código abaixo no console (F12):

var chatData = document.querySelector('#main header')?.parentElement?.parentElement?.__reactFiber$?.return?.memoizedProps?.children?.props?.chat; console.log("--- DADOS PARA O SEU BOT ---"); console.log("Nome:", chatData?.name); console.log("ID para copiar:", chatData?.id._serialized); console.log("----------------------------");
Importante: Grupos terminam em @g.us e Contatos em @c.us. Não use o ID da mensagem (A5F8...), use o ID da conversa acima.
